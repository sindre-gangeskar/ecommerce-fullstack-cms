# Fullstack E-commerce with CMS (Next.js Payload) - Under Development  

This project entails the full-stack flow and the usage of CMS with Next.js Payload which allows for full GUI support for a full *pretend* ecommerce project.

### About
This is still under development, but its purpose is to showcase an understanding of full-stack development with the typical express.js as backend + SQL database interactions with react in frontend, and the usage of Cloudflare for images served as well as a headless CMS which is meant to allow for easy creation of new products etc.

It will be hosted as a live project once it's done.
